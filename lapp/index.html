Conclusion & Perspectives

- **Ultimate goal: Create an event map to localise gamma-ray sources** - Currently trying to reduce the discrepancy between MC and real data (unbalanced representation of the classes) - **Working in parallel on:** - Replace MC+P($\lambda)$ with real data (Markarian 501, Crab, ...) - The pointing direction between training data and observations - Introduction of physics knowledge (Physics-based deep learning) - **Future challenges:** - Stereoscopy ([Federated learning](https://arxiv.org/abs/1902.04885), ...) - Neural network explanability ([Information bottleneck](https://arxiv.org/abs/1503.02406), ...) - Sensor drifs ([Continual learning](https://arxiv.org/abs/1909.08383), ...)
